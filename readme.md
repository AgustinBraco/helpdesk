Run server:
`python3 manage.py runserver`

URLs:
```
.../admin
.../users
.../tickets
```

Python venv:
`python3 -m venv env`

Start app:
`python3 manage.py startapp name apps/name`

Make migrations:
`python3 manage.py makemigrations`

Migrate:
`python3 manage.py migrate`

Create super user:
`python3 manage.py createsuperuser`
